# 空き時間抽出アプリ - ユーザーガイド

## 概要

**目的:** Googleカレンダーと連携して、指定期間内の空き時間を自動抽出し、コピー可能なテキスト形式で出力します。

**アクセス:** ログインが必要です。

## Powered by Manus

このアプリケーションは最新のWeb技術で構築されています。

**フロントエンド:** React 19 + TypeScript + Tailwind CSS 4 + shadcn/ui コンポーネントライブラリによる洗練されたUIを実現しています。

**バックエンド:** Express 4 + tRPC 11による型安全なAPI通信により、フロントエンドとバックエンド間のデータのやり取りが完全に型チェックされ、実行時エラーを最小限に抑えています。

**データベース:** MySQL/TiDB + Drizzle ORMによる高速で信頼性の高いデータ管理を実現しています。

**認証:** Manus OAuth + Google OAuth 2.0による安全なユーザー認証とGoogleカレンダーへのアクセスを提供します。

**デプロイメント:** Auto-scaling infrastructure with global CDN

## Webサイトの使い方

アプリケーションを開くと、まずGoogleカレンダーとの連携が必要です。"Googleカレンダーと連携"ボタンをクリックすると、Googleのログイン画面が表示されます。アカウントを選択し、カレンダーへのアクセスを許可してください。連携が完了すると、"連携済み"と表示されます。

次に、空き時間を確認したい期間を設定します。"開始日"と"終了日"の入力欄に日付を入力してください。デフォルトでは今日から1週間後までが設定されています。また、"開始時刻"と"終了時刻"で1日の中で空き時間を探す時間帯を指定できます(デフォルトは9時から18時)。"枠の長さ(分)"では、抽出する空き時間の最小単位を設定します(デフォルトは30分)。

設定が完了したら、"空き時間を確認"ボタンをクリックします。アプリがGoogleカレンダーからイベント情報を取得し、空き時間を計算します。結果は日付ごとにグループ化されたリスト形式で表示されます。

表示された空き時間は、"コピー"ボタンをクリックすることで、クリップボードにコピーできます。コピーしたテキストは、メールやチャットなどに貼り付けて、相手に空き時間を共有できます。

## Webサイトの管理

このアプリケーションの設定は、管理UIから変更できます。

**Settings パネル:** アプリケーション名やロゴを変更できます。また、Google OAuth認証情報(GOOGLE_CLIENT_ID、GOOGLE_CLIENT_SECRET)はSecretsセクションで管理されています。

**Database パネル:** ユーザー情報やGoogleカレンダーのトークン情報を確認・編集できます。

**Dashboard パネル:** アプリケーションのアクセス統計を確認できます。

## 次のステップ

Manus AIにいつでも話しかけて、機能の追加や変更をリクエストできます。例えば、特定の曜日だけを対象にする機能や、複数のカレンダーを統合する機能などを追加できます。

### 本番環境への準備

現在、Google OAuth認証情報はテスト用の設定になっている可能性があります。本番環境で公開する前に、以下を確認してください。

**Google Cloud Console:** Settings → Secrets で GOOGLE_CLIENT_ID と GOOGLE_CLIENT_SECRET が本番用の値に設定されていることを確認してください。Google Cloud Consoleで本番用のOAuth 2.0クライアントを作成し、承認済みのリダイレクトURIに本番環境のURLを追加してください。

本番環境で公開する前に、Google Cloud Consoleから本番用の認証情報を取得してください。
