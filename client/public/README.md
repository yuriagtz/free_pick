# 画像ファイルの保存場所

このディレクトリ（`client/public`）に以下の画像ファイルを保存してください：

## 必要な画像ファイル

1. **ロゴ画像**: `FreePick_logo.png`
   - ブラウザのタブやアプリ内で表示されるメインロゴ
   - 推奨サイズ: 512x512px または 1024x1024px（高解像度対応）

2. **ファビコン**: `favicon.ico`
   - ブラウザのタブアイコンとして表示
   - 推奨サイズ: 32x32px、64x64px、または 128x128px（複数のサイズにも対応可能）
   - 形式: ICO形式（推奨）または PNG形式

## ファイル名の要件

- ロゴ画像は `/FreePick_logo.png` として参照されます（`client/src/const.ts`で設定）
- ファビコンは `/favicon.ico` として参照されます（`client/index.html`で設定）

## 保存後の動作

- 開発環境: `client/public` のファイルは自動的に `/` ルートで提供されます
- 本番環境（Vercel）: ビルド時に `dist/public` にコピーされ、静的ファイルとして配信されます

## 注意事項

- ファイル名は大文字・小文字を区別します
- 日本語ファイル名は避けてください
- 画像ファイルが存在しない場合、デフォルトのプレースホルダーが表示されます
